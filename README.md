# d3Model
使用d3实现模型例子构建

# 功能详情：
1. 自由拖动，缩放，移动，连线。
2. 选择，单击，右键操作
3. 拖动合并等

# 启动方式：
进入项目根目录打开控制台运行以下命令
1. npm install 或 yarn install (安装依赖)
2. npm start (启动)
3. npm run build (打包)
4. 本地打开：http://localhost:9090 即可

! 注意：node版本：12.16.2
